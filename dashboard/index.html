TradingView Alert Templates

Add the Argus Swing Pine Script indicator to your SPY, SPXW, IWM and QQQ charts on the 30-minute timeframe. Create one alert per ticker using these conditions:

Bar Close — SPY (repeat for each ticker)
{"ticker":"SPY","event":"bar_close","close":{{close}},"sma55":{{plot("55 SMA")}},"option_price":{{plot("Option Price")}}}
Weekly Expected Move Hit — SPY
{"ticker":"SPY","event":"weekly_move_hit","option_price":{{plot("Option Price")}}}

⚠ Replace SPY with SPXW, IWM, or QQQ for each chart. Set trigger to Once Per Bar Close.

Ticker Settings

Enable or disable each ticker. Disabled tickers will be ignored even if alerts fire.
